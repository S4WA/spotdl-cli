[ How to Install & Download ]

 1:  Install Python 3
 2:  Run install.bat
 3:  Run download.bat first & Open CONFIG.INI
 4:  Put your spotify client id/secret in CONFIG.INI
 5:  Put the links of song/album/playlist you want to 
    download in DOWNLOAD-LIST.TXT
    (One link per line should be placed.)
 6:  Run download.bat again
 7:  Donwloaded files will be in the "output" folder.
